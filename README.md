# 🎵 MusicHub - 音樂串流網站

MusicHub 是一個現代化的音樂串流應用，模仿 Spotify 的設計和功能。

## 功能特性

- ✨ **現代化界面** - 採用 Spotify 的深色設計風格
- 🎵 **播放器控制** - 播放、暫停、上一首、下一首功能
- ❤️ **音樂庫管理** - 將喜歡的歌曲保存到個人音樂庫
- 🔍 **搜尋功能** - 快速搜尋歌曲和藝人
- 📊 **播放進度** - 實時顯示播放時間和總時長
- 🔊 **音量控制** - 調整播放音量

## 快速開始

1. 在瀏覽器中打開 `index.html`
2. 瀏覽歌曲列表並選擇播放
3. 使用播放器控制歌曲
4. 點擊 ❤️ 將喜歡的歌曲添加到音樂庫

## 使用的技術

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- LocalStorage (用於保存用戶偏好設定)

## 功能說明

### 播放器
- **播放/暫停** - 點擊大的播放按鈕
- **上一首/下一首** - 使用側邊的按鈕
- **進度調整** - 點擊進度條上的任意位置

### 音樂庫
- 在歌曲卡片上點擊 ❤️ 來添加到音樂庫
- 在 "我的音樂庫" 部分查看所有保存的歌曲
- 音樂庫數據保存在本地瀏覽器中
