# VS Code 和 Git 学习项目

这是一个用于学习 VS Code 编辑器和 Git 版本控制的示例项目。

## 项目结构

```
.
├── src/
│   ├── index.js      # 主程序
│   └── utils.js      # 工具函数
├── package.json      # 项目配置
├── .gitignore        # Git 忽略文件
└── README.md         # 项目文档
```

## 如何运行

```bash
node src/index.js
```

## Git 学习内容

- 版本控制基础
- 提交历史查看
- 分支管理
- 上传到远程仓库
