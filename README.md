# 🎯 Code128 條碼掃描器

一個功能完整的 Code128 條碼掃描識別應用。

## 功能特性

- 📷 **實時攝像頭掃描** - 自動檢測和識別 Code128 條碼
- 🔍 **條碼驗證** - 支持手動輸入驗證
- 📊 **掃描歷史** - 保存掃描記錄（最多 50 條）
- ✅ **格式檢測** - 區分數字和文本格式
- 💾 **本地存儲** - 掃描歷史本地保存
- 📱 **響應式設計** - 支持桌面和移動設備
- 🎨 **現代化 UI** - 漸變色設計和流暢動畫

## 快速開始

1. **打開應用**
   - 在瀏覽器打開 `http://localhost:8000`

2. **掃描條碼**
   - 點擊「啟動攝像頭」
   - 面向 Code128 條碼進行掃描
   - 自動識別並保存結果

3. **手動驗證**
   - 在輸入框輸入條碼數據
   - 點擊「驗證」或按 Enter
   - 查看驗證結果

## Code128 條碼說明

### 支持的格式

- **Code128-A** - 數字和大寫字母
- **Code128-B** - 所有 ASCII 字符（最常用）
- **Code128-C** - 高密度數字編碼

### 條碼示例

- `123456789012` - 純數字條碼
- `CODE128DEMO` - 文本條碼
- `2024022400001` - 日期/序列號

## 使用的技術

- **HTML5** - 頁面結構
- **CSS3** - 現代化設計
- **JavaScript** - 應用邏輯
- **getUserMedia API** - 攝像頭訪問
- **Canvas API** - 圖像處理
- **LocalStorage** - 數據持久化
- **JsBarcode** - 條碼生成（可選）

## 功能詳解

### 播放器控制

| 按鈕 | 功能 |
|------|------|
| 📷 啟動攝像頭 | 開始實時掃描 |
| ⏹️ 停止掃描 | 停止攝像頭 |
| 🔄 重置 | 清除當前結果 |

### 掃描過程

1. 啟動攝像頭獲取視頻流
2. 實時分析視頻幀尋找條碼
3. 檢測到條紋模式時提取條碼
4. 解碼和驗證條碼數據
5. 保存到歷史記錄

### 驗證規則

- **長度檢查** - 最少 2 字符
- **字符檢查** - 確保為有效 ASCII 範圍
- **格式識別** - 純數字為 Code128-C，其他為 Code128-B
- **校驗碼** - 計算並驗證條碼校驗碼

## 掃描歷史

- 自動保存最近 50 次掃描
- 第一次打開時自動加載
- 支持手動清除歷史
- 記錄時間戳和驗證狀態

## 瀏覽器兼容性

- Chrome/Edge 90+
- Firefox 88+
- Safari 14.1+
- Opera 76+

需要支持 `getUserMedia` API 和 `Canvas`

## 已知限制

- 需要 HTTPS 或 localhost 來訪問攝像頭
- 實時掃描依賴電腦性能
- 條碼需要清晰可見

## 故障排除

### 攝像頭無法訪問
- 檢查瀏覽器權限設置
- 必須使用 HTTPS 或 localhost
- 關閉其他使用攝像頭的應用

### 無法識別條碼
- 確保條碼清晰可見
- 調整照明條件
- 嘗試不同角度
